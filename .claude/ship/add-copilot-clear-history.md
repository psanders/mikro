## Decision log

- 2026-07-04 — Checkpoint created. Draft proposal/spec/tasks written from user's initial idea (small button, soft-delete internally, clear the view). Open design question surfaced: what happens to an unresolved pending write action when history is cleared — leaning "block the clear" over "silently reject the action" but needs user confirmation.
- 2026-07-04 — Noted pre-existing uncommitted diff in mods/dashboard/src/founder/copilot/ (CopilotDock composer autosize/autoscroll, header font-weight) is unrelated in-flight polish from a prior session, not part of this change's scope — left untouched.
- 2026-07-04 — Two repo gotchas hit during build, worth remembering for future schema changes: (1) `@mikro/common` re-exports schemas by explicit name in TWO places (`src/schemas/index.ts` AND the root `src/index.ts` barrel) — adding a schema requires updating both plus `npm run build` in mods/common, or consumers get "no exported member" even though the file itself is correct. (2) `mods/apiserver/test/integration/setup.ts` hardcodes a `SCHEMA_SQL` string for the in-memory test DB, entirely separate from `prisma/migrations/` — a new column needs manual duplication there too, or integration tests fail with "column does not exist" despite the real migration being correct.
- 2026-07-04 — Build + Test stages complete, all gates green. Entering Sync (human gate) — awaiting user approval before promoting the delta spec into openspec/specs/founder-copilot.
